NINJA PLAYER DATA v36

【変更】
・「いいところ」「改善点」「方向性」をスプレッドシートへ保存
・新しいシート「選手フィードバック」を自動作成
・選手IDごとに1行保存し、再保存時は同じ行を更新
・別端末、別ブラウザでも同じ内容を取得可能
・既存の端末内保存データは、保存ボタンを押すとスプレッドシートへ移行
・通信失敗時は端末にも一時バックアップ

【必須更新】
1. Apps Scriptの Code.gs をv36へ全置換
2. Apps Scriptで「新しいデプロイ」を作成
3. 発行された /exec URLをアプリの「接続設定」へ入力
4. GitHubへ index.html / manifest.json / sw.js / icon.svg を上書き
5. Command + Shift + Rで再読み込み

【スプレッドシートに作成される列】
選手ID / 選手名 / カテゴリー / いいところ / 改善点 / 方向性 / 更新日時
