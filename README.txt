NINJA PLAYER DATA 試作版

【内容】
・選手一覧／カテゴリー検索
・選手別ページ
・身長：初回との比較、直近3か月グラフ、全履歴
・体重：初回との比較、直近3か月グラフ、全履歴
・シューティング：総数、種目別、ポジション別
・アジリティ：最新値、前回比、履歴
・コーチ所見（試作版は端末内保存）
・PWA対応

【接続方法】
1. index.htmlを開く
2. 右上「接続設定」
3. 現在のシューティングアプリ用Apps Script WebアプリURLを貼る
4. 「保存して接続」

既存Apps Script v11.3の以下APIを使用します。
listPlayers / playerDetail / growthRecords / agilityRecords

【GitHub Pages】
このフォルダ内のファイルをリポジトリ直下へアップロードし、
Settings → Pages → Deploy from a branch → main / root を選択します。

【注意】
・Apps Scriptは「ウェブアプリ」としてデプロイし、アクセス権を適切に設定してください。
・未成年者の身体データを扱うため、正式運用時は認証を追加してください。
・Chart.jsはCDNから読み込みます。初回表示にはインターネット接続が必要です。

【v10 AIフィードバック3項目】
AI所見とは別に、次の3項目を追加しました。

・いいところ
  選手の強み、成長、継続できている点を表示

・改善点
  次に伸ばしたい具体的な課題を表示

・方向性
  練習や試合で何を意識し、どのように伸ばすかを表示

各項目はAI生成後にコーチが自由に修正できます。
「3項目を保存」で選手ごとに端末内へ保存します。

【Apps Scriptへの追加】
既存doGet(e)のaction分岐へ以下を追加してください。

case 'generatePlayerFeedback':
  return jsonp_(e, generatePlayerFeedback_(e.parameter.payload));

google_apps_script_ai_addon.js の generatePlayerFeedback_ もApps Scriptへ追加してください。
